# vue-game-cli

## 基本功能

- 通过命令行方式进行游戏配置

## 可用指令
* 初始化项目目录
- vue-game init <project name>

* 构建游戏
- vue-game create <template name>

* 更新webpack配置文件
- vue-game update

* 生成resource.json文件
- vue-game resource <template name>


## 相关npm包

- commander   自定义命令行
- inquirer    命令行交互工具
- chalk       控制台多颜色输出
- ora         控制台进度条
- log-symbols 日志级别的彩色符号
